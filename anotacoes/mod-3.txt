ASSUNTO DE SCOPO ABAIXO: [ASSUNTO IMPORTANTE]
Para que saver os scopos ? Os scopos ajuda os navegadores a identificar melhor a sua tabela, quantas colunas, linhas e etc tem nela.

scope="col" >>> Use esse Scope para definir Ths que estão em cima das colunas, ou seja eles estão como se fossem títulos
scope="colgroup" >>> Esse Scope tem a mesma função do de cima, porém esse pega para todas as colunas, por isso é colgroup, o de cima é apenas para uma coluna, (Use esse sempre que o Th ocupar mais de uma coluna)
scope="row" >>> Scope para linhas, ou seja na frente do Th tem dados, (Tds), então quando estiver em linha use row
scope="rowgroup" >>> Tem a mesma função do row, porem esse pega varias linhas, o de cima pega apenas uma linha
